Changing an existing video effect

You can make changes to a video effect that has already been rendered.

To change an effect:

  1. Select the clip that contains the effect you want to change in the clip viewer or timeline viewer.

    2. Tip: If you need to make the same change to more than one clip, select all the clips and update them all at once to save some time. For information about selecting multiple clips, see Related Topics below.

  2. Press the Delete key to remove the effect.
  3. Click the Editing button, and then click Video FX at the top of the Editing pane.
  4. Set your new effect settings.
  5. Click Apply.
