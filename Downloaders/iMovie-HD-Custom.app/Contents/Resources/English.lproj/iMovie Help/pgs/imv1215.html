Using iWeb with your HomePage sites

If you have previously published movies to .Mac using the HomePage pane in a version of iMovie earlier than iMovie HD 6, those webpages are still available at:

http://homepage.mac.com/YourMemberName

Websites you create in iWeb are available at:

http://web.mac.com/YourMemberName/iWeb/SiteName

On your iDisk, your HomePage sites are stored in the Sites folder. Your iWeb sites are stored in the Sites folder inside the Web folder (Web/Sites).

You can't edit HomePage sites in iWeb, but you can set up links on your HomePage sites to go to your iWeb sites, and vice versa.

To learn how to link to a HomePage site from an iWeb site, see iWeb Help, available in the Help menu when iWeb is open.

To link to an iWeb site from a HomePage site, add the link to the site menu for your HomePage site.

To set up a site menu page using HomePage:

  1. Go to www.mac.com and click HomePage on the .Mac tab.
  2. Select the HomePage site you want to add a menu to.
  3. Click the Site Menu tab, and then choose a page.
  4. Click the Add (+) button to add another menu item to the page.
  5. Click the Edit Link button for the new menu item, type the address of the iWeb site you want to link to, and click apply.
  6. When you're ready to publish the page to your HomePage site, click Publish.

You can also use iWeb to republish movies to the web that you previously published using HomePage. Just use iMovie HD 6 to export the movies directly into iWeb and publish them to .Mac. After you republish the movies using iWeb, be sure to delete them from your HomePage sites in order to conserve your available iDisk storage. For more information, see "Deleting a page or site" in .Mac Help. (To see .Mac Help, go to www.mac.com and click Help on the .Mac tab.)
