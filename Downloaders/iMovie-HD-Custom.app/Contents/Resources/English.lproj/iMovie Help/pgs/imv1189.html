Reverting to the last saved version of a project

After making changes to a project, if you decide that you want to start over, you can revert to the last saved version of the project. iMovie discards all the changes you made since the last time you saved.

To revert to the last saved version of a project:

Choose File > "Revert to Saved."

The Revert command is dimmed if you haven't made any changes to the project.

You can also undo your recent changes by choosing Edit > Undo.
