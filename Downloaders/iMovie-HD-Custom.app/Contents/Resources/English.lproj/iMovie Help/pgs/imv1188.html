Saving changes

The changes you make to your movie aren't permanent until you save the project. To avoid losing changes, you should save your work frequently.

WARNING: After you save your project, you cannot undo any of the changes you made to your project before that time.

To save changes to your project:

Choose File > Save Project.

If you want to create different versions of the same project, you can save a copy of your project and give it a new name. Choose File > Save Project As and give it a new name. After you save, the project with the new name remains open. To work with the previous version, you must reopen it. Choose File > Open Recent and choose the previous project from the submenu.
