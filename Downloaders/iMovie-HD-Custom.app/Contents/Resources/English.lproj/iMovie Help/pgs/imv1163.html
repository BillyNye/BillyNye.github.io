Capturing a still image from your video

You can select a frame in your movie and save it as a still image. You can then add it to your movie and use it as you would any photograph. You can also save the still image as a file to export to another application. You might then email it to a friend, or edit the image in another application and import the revised image back into the movie later.

To create a still image from video:

  1. Drag the playhead (shown above) to the frame you want to capture.

    2. Tip: To precisely adjust the playhead, press the Left or Right Arrow key to move it one frame at a time. To move it in ten-frame increments, hold down the Shift key while pressing the arrow key.

  2. Choose Edit > Create Still Frame.

    3. iMovie places the new clip of the still image in the Clips pane (or at the end of the clip viewer or timeline viewer if you set your preferences to import clips there).

To save the image as a file to export to another application, choose File > Save Frame in step 2 above, then save the file with the format and name you want. You can save still images in JPEG or PICT file format. If you intend to send an image through email or post it on a webpage, choose JPEG. If you plan to reuse the still image in your iMovie project, choose PICT.

Note: You can't apply pan and zoom effects to still images you create from digital video.
