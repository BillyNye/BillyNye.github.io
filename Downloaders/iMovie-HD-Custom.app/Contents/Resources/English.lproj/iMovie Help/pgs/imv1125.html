I see a message that the disk responded slowly

If an error message says, "The disk responded slowly," there may be a problem with disk fragmentation. Frequent creation and deletion of files can cause disk fragmentation, which means that available disk space is not found in contiguous blocks. This is not a serious problem, but it causes the disk to respond more slowly while you are importing media.

If you see this error message, you can use a third-party disk defragmentation utility to defragment your hard disk. You can also open Disk Utility, located in /Applications/Utilities, and run First Aid on your hard disk.

If you are using the FileVault security feature of Mac OS X, you might turn it off or move your iMovie HD project to a location outside your home folder while you are working with iMovie HD. A good location is the Shared folder, located in the Users folder on your hard disk.
