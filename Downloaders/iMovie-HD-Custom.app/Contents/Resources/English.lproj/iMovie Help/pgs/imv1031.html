Renaming clips

iMovie HD gives each clip a generic name (Clip 01, Clip 02, and so on). It's a good idea to change the name to something that is more meaningful to you. Using understandable clip names can help speed up the editing process because you can more easily identify clips you're looking for without searching or reviewing each clip.

Here are ways to rename a clip:

  • Double-click the clip and type a new name in the Clip Info window.
  • Select the clip in the clip viewer or Clips pane, click the clip's name, and then type a new one.
