Which audio formats are compatible with iMovie HD?

You can import audio in any format that works with QuickTime. Two of the more common audio formats are:

  • AIFF (Audio Interchange File Format), a digital file format used by iMovie HD and many other applications
  • MP3 (MPEG-1 layer 3), a digital-format audio file that can be used by iMovie HD and many other applications

Tip: For the best compatibility with iMovie HD transitions, effects, titles, and the Ken Burns Effect, you should record your video with 48 kilohertz (kHz), 16-bit audio. Check your camera's manual for instructions on setting the audio bit rate and depth.
