Saving different versions of the same project

If you want to create different versions of the same project, you can save a copy of your project and give it a new name.

Note: Saving multiple copies of a project can use large amounts of disk space.

To create different versions of the same project:

  1. Choose File > Save Project As.
  2. Type a new name and choose a location for the copy of the project.
  3. Click Save.

After you save, the project with the new name remains open. To work with the previous version, you must reopen it. Choose File > Open Recent and choose the previous project from the submenu.
