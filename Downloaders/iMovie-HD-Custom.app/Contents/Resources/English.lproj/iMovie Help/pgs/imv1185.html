Determining which clips contain a title or effect

iMovie labels any clips that you've added a title or effect to. This makes it easy to find any title or effect you want to change or remove.

To see which clips contain a title or effect:

  1. Click the Clip Viewer button (shown above).
  2. Look along the top edge of each clip for the following icons:
    • If a clip contains an effect, iMovie labels it with an effect icon (A, shown below). If a clip contains more than one effect, iMovie displays the number of effects to the left of the effect icon.
    • If a clip contains a title, iMovie labels it with a title icon (B, shown below), and the title appears on the bottom edge of the clip.
